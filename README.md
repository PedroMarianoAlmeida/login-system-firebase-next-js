This is a Next.js + Material UI + Prettier + TypeScript Project

## Install dependencies

```bash
npm install
```

## Getting Started

First, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
